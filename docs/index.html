Package vcodeboot

Bibliothèque Python pour le traitement et la manipulation d'images

Présentation

vcodeboot est un package Python conçu pour faciliter la manipulation et le traitement d'images. Il offre des outils puissants et flexibles pour travailler avec des images pixel par pixel.

Modules disponibles

vcodeboot.globaux

Module contenant les variables et fonctions globales utilisées dans le package. Fournit les fondations pour les opérations de traitement d'image.

vcodeboot.pixels

Module principal pour la manipulation des pixels et le traitement d'image. Contient les fonctions essentielles pour la transformation et l'analyse d'images.

Installation

Pour installer vcodeboot, utilisez pip :

pip install vcodeboot
