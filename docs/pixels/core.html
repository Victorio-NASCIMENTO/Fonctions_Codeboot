def getMouse()

Gets the mouse position and state, along with modifier key states (Alt, Ctrl, Shift).

This function captures mouse events and returns its coordinates, left button state, and information about Alt, Ctrl, and Shift keys. Depending on the turtle_mode parameter, it adapts the display method using either the pygame library (for classic event handling) or simulating the Turtle environment with Tkinter if pygame fails.

Returns

struct
An object containing mouse coordinates (x, y), button state (left click active or not), and Alt, Ctrl, and Shift key states.
